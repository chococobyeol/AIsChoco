# TTS 변환 속도

## Qwen3-TTS에서 쓸 수 있는 경량 모델

- **Qwen3-TTS-12Hz-0.6B-CustomVoice** (현재 사용 중) = 공식 **최소** 모델 (약 0.6B 파라미터, VRAM 약 2GB).
- 1.7B는 더 크고 느리므로, **Qwen3-TTS 계열에서는 0.6B보다 작은 모델이 없습니다.**

## Qwen3-TTS 속도 올리는 방법

1. **FlashAttention 2** (선택)  
   추론 속도·VRAM 개선 가능하지만, Windows에서는 빌드 실패가 잦아 **설치하지 않고 사용해도 됨**. 없어도 Qwen3-TTS 정상 동작.

2. **GPU 사용**  
   CUDA + cu126 등 PyTorch CUDA 빌드 사용 (이미 적용된 상태면 유지).

3. **스트리밍**  
   Qwen3-TTS는 스트리밍 생성(첫 패킷 지연 ~97ms)을 지원하지만, `qwen-tts` Python API에서 스트리밍 옵션이 어떻게 노출되는지는 패키지/버전별로 확인 필요.

## 더 가벼운 TTS 대안 (경량·고속)

| 엔진        | 규모        | 특징                          | 한국어 |
|------------|-------------|-------------------------------|--------|
| **Piper**  | 약 5~32M 파라미터 | 로컬, 매우 빠름, 오프라인       | [piper-voices](https://huggingface.co/rhasspy/piper-voices)에서 한국어 보이스 여부 확인 |
| **edge-tts** | 클라우드   | 무료, 저지연                  | 지원   |
| **gTTS**   | 클라우드   | 매우 단순                     | 지원 (품질 제한) |

실시간 채팅 반응용으로 **속도 우선**이면 Piper(한국어 보이스 있을 때) 또는 edge-tts를 같이 검토할 수 있습니다. **음질·감정 표현 우선**이면 Qwen3-TTS 0.6B + FlashAttention 유지가 적합합니다.
